Provide a simple widget form to search for amateur radio call signs on QRZ.com.

== Description ==
This widget plugin is directed primarily toward amateur radio enthusiasts with 
WordPress blogs. It provides a form to lookup amateur radio
[call signs](http://wikipedia.com/Call_sign)
from the [QRZ.com site](http://qrz.com/). QRZ.com provides information
about operators indexed by their call sign.

== Installation ==
This widget plugin has been tested on WordPress 2.9.2 and should work
on previous 2.x versions as well. PHP 5.x is required.

= Manual Install =

1. Add a directory called 'qrz-search' (without the quotes) to your 
   '/wp-content/plugins/' directory.
1. Upload qrzsearch.php to '/wp-content/plugins/qrz-search/'
   directory.
1. Activate the plugin through the 'Manage Plugins' page in WordPress.
1. Place the widget in the sidebar using the editor for your theme.

= Automatic Install =

1. Download the [qrzsearch.zip](http://ns6t.net/qrzsearch/qrzsearch.zip) file.
1. Go to the Manage Plugins page.
1. Click the Add New button.
1. Upload the [qrzsearch.zip](http://ns6t.net/qrzsearch/qrzsearch.zip) file.
1. Active the plugin through the 'Manage Plugins' page.
1. Place the widget in the sidebar using the editor for your theme.

== Frequently Asked Questions ==

= What is QRZ.com? =

It's a popular ham radio site where ham's lookup information about operator's
based on their call sign. It also has lots of other ham radio information.

= Am I allowed to do this? =

QRZ.com [welcomes](http://www.qrz.com/i/linking.html) all sites to add
searches like this.
